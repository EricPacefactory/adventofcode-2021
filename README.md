# Advent of Code 2021

Solutions for 2021 [advent of code](https://adventofcode.com/)

---

Using [Julia](https://julialang.org/) (1.6.4) and/or [Python3](https://www.python.org/) (3.8.10, but 3.6+ should work)

All solutions are aiming to avoid external libraries, so no special environments are needed.
However, all scripts are set up to load input data using relative pathing, so they must be run from within the day folders!
